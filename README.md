# Malwaro - Loja de Games e Produtos Eletrônicos

## Visão Geral do Projeto

Este projeto é uma loja online de games e produtos eletrônicos, desenvolvido originalmente como projeto final de faculdade no ano de 2021. Em 2026, o projeto está sendo atualizado para incorporar novas funcionalidades e melhorias.

## Tecnologias Utilizadas

O projeto Malwaro é construído utilizando as seguintes tecnologias, frameworks e bibliotecas:

### Backend

*   **ASP.NET Core 8.0:** Framework de código aberto e multiplataforma para a construção de aplicações web modernas e APIs.
*   **C#:** Linguagem de programação principal utilizada no desenvolvimento do backend.
*   **Entity Framework Core 8.0:** ORM (Object-Relational Mapper) que facilita a interação com o banco de dados, permitindo que os desenvolvedores trabalhem com objetos C# em vez de consultas SQL diretas.
*   **Microsoft.AspNetCore.Identity.EntityFrameworkCore:** Biblioteca para gerenciar autenticação e autorização de usuários, incluindo registro, login e gerenciamento de perfis.

### Banco de Dados

*   **SQLite:** Banco de dados leve e embarcado, utilizado para armazenamento de dados no projeto.

### Frontend

*   **HTML5:** Linguagem de marcação para estruturar o conteúdo das páginas web.
*   **CSS3:** Linguagem de estilo utilizada para a apresentação visual das páginas.
*   **JavaScript:** Linguagem de programação para adicionar interatividade e dinamismo ao frontend.
*   **Bootstrap:** Framework de código aberto para desenvolvimento responsivo e mobile-first, fornecendo componentes de UI pré-estilizados.
*   **jQuery:** Biblioteca JavaScript rápida, pequena e rica em recursos que simplifica a travessia e manipulação de documentos HTML, tratamento de eventos, animação e interações Ajax.
*   **jQuery Validation:** Plugin jQuery para validação de formulários no lado do cliente.
*   **jQuery Validation Unobtrusive:** Adaptação do jQuery Validation para funcionar de forma não intrusiva com ASP.NET Core, utilizando atributos de dados HTML5 para as regras de validação.

## Funcionalidades Principais

*   Catálogo de produtos com detalhes.
*   Sistema de carrinho de compras.
*   Processo de checkout com opções de pagamento (PIX e Cartão de Crédito).
*   Sistema de parcelamento para pagamentos com cartão de crédito (até 5x sem juros, acima de 5x com juros de 1,5%).
*   Autenticação e autorização de usuários.
*   Gerenciamento de pedidos.
